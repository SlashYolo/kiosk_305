# МАИ Стенд — Информационный киоск кафедры 305

Сенсорный TV-стенд для отображения расписания МАИ, информации о кафедре 305 и её преподавателях. Разработан для работы на Lubuntu с сенсорным экраном.

---

## Структура проекта

```
mai-kiosk/
├── mai_kiosk.html          # Основное приложение (SPA)
├── server.py               # Локальный прокси-сервер (Flask)
├── mai_schedule_parser.py  # Парсер API расписания МАИ
├── cache_schedule.py       # Скачивание всего расписания в кэш
├── wifi_schedule_update.py # Плановое обновление кэша через Wi-Fi
├── start.sh                # Запуск приложения (для разработки)
├── mai-kiosk.sh            # Автозапуск на стенде (Lubuntu)
├── afk-video.sh            # AFK-заставка (видео при бездействии)
├── hide-panel.sh           # Скрытие панели LXQt для киоск-режима
├── install.sh              # Установка всего на Lubuntu одной командой
├── requirements.txt        # Python-зависимости
├── on.mp4                  # Видео-заставка при бездействии (добавить вручную)
├── schedule_cache/         # Кэш расписания (создаётся автоматически)
└── pubs_cache/             # Кэш публикаций преподавателей (создаётся автоматически)
```

## Быстрый старт

### Требования
- Python 3.10+
- Chromium или Google Chrome
- Lubuntu / Ubuntu (для стенда)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск (разработка / macOS / Windows)

```bash
./start.sh
# Откроется браузер на http://localhost:8765
```

### Установка на стенд Lubuntu (один раз)

```bash
bash install.sh
sudo reboot
```

После перезагрузки стенд запускается автоматически в полноэкранном режиме.

## Архитектура

```
Браузер (mai_kiosk.html)
        │
        │  HTTP  localhost:8765
        ▼
  server.py  (Flask прокси)
        │
        ├──► maiapp.lavafrai.ru/api/v1  (расписание)
        ├──► mai.ru/education/...       (публикации преподавателей)
        └──► schedule_cache/            (офлайн-кэш)
```

Браузер не может напрямую обращаться к MAI API из-за CORS-ограничений, поэтому все запросы проходят через локальный Flask-сервер.

## Функциональность

### Вкладки приложения

| Вкладка | Описание |
|---|---|
| **Расписание** | Поиск по группе или преподавателю. Онлайн + офлайн-режим |
| **О кафедре** | Описание кафедры 305, научные школы, лаборатории, контакты |
| **Преподаватели** | Список сотрудников кафедры 305 с публикациями |
| **Программы обучения** | Специалитет 24.05.06 |
| **ЧаВо** | Ответы на частые вопросы пользователей стенда |

### Офлайн-режим

При потере соединения приложение автоматически переключается на кэш:
- 5 секунд без сети → переход в офлайн (красный статус-бар)
- Данные берутся из `schedule_cache/` и `pubs_cache/`
- При восстановлении соединения — 10 секунд на плавный переход обратно

### Кэширование расписания

```bash
# Скачать всё расписание вручную (~7-8 минут, ~150 МБ)
python3 cache_schedule.py

# Автоматически в 10:00 каждый день (добавить в cron)
crontab -e
# → 0 10 * * * /usr/bin/python3 /home/user/mai-kiosk/cache_schedule.py >> /home/user/mai-kiosk/cache.log 2>&1
```

### Плановое обновление через Wi-Fi (Lubuntu)

```bash
# Первичная настройка (один раз)
python3 wifi_schedule_update.py --setup
# Введёт SSID и пароль, зашифрует по machine-id устройства

# Добавить в cron (выполняется в 10:00)
# 0 10 * * * python3 ~/mai-kiosk/wifi_schedule_update.py
```

## Горячие клавиши

| Сочетание | Действие |
|---|---|
| `Shift → Tab → Space` | Включить/выключить сенсорную клавиатуру |
| `Ctrl + Shift + Alt + A + E` | Выход из приложения (с подтверждением) |
| `Ctrl + Shift + Alt + F` | Выход из полноэкранного режима |
| `Escape` | Закрыть модальное окно |

Шорткаты настраиваются в массиве `SHORTCUTS` в `mai_kiosk.html`. Чтобы отключить — `enabled: false`.

## AFK-видео заставка

Положи файл `on.mp4` в папку проекта. После 5 минут бездействия автоматически запускается в полноэкранном режиме. Любое касание экрана — видео закрывается.

```bash
# Проверить что xprintidle и mpv установлены
sudo apt install xprintidle mpv
```

## API сервера

| Эндпоинт | Описание |
|---|---|
| `GET /` | Открывает mai_kiosk.html |
| `GET /api/health` | Статус сервера |
| `GET /api/groups` | Список всех групп МАИ |
| `GET /api/teachers` | Список преподавателей |
| `GET /api/schedule/<name>` | Расписание группы или преподавателя |
| `GET /api/teacher-pubs/<guid>` | Публикации преподавателя |
| `GET /api/cache-status` | Дата и статистика последнего обновления кэша |
| `GET /api/cache-names?type=group\|teacher` | Имена из офлайн-кэша (для подсказок) |
| `GET /api/debug-pubs/<guid>` | Диагностика страницы профиля преподавателя |
